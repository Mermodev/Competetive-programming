## What is this repository for
In this repository I store my solutions to tasks on multiple sites such as [CSES](https://cses.fi/), [sio2](https://sio2.mimuw.edu.pl/c/oi33-1/dashboard/) or [codeforces](https://codeforces.com/).
Some of the tasks may not be available publically. In that case I will try to add the task and a checking program(or you can create one yourself) to the repository so that you too can try to solve the task.
Apart from them you can also find things like my c++ headers, compilation aliases, vimrc and bashrc configuration and many more things that can be usefull with competetive programming.
Disclaimer: I am an ammateur

## Are the submitted solutions correct?
The solutions that are not specifically marked (either by having name in a different format than task_name.cpp and/or having it written in the first line of the code) as not working pass all the tests on the site they were submitted to.
Even though all submissions are correct this does not mean that they are the best, it simply means that they are as good as they need to be to pass.
